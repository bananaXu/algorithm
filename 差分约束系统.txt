差分约束系统：
求解
x1 - x2 <= a1;
x3 - x2 <= a2;
x2 - x4 <= a3;
x4 - x3 <= a4;
...
xi-xj <= an;类似最短路中d[v] <= d[u]+w[u, v];
因此可以将差分约束转换为求图的最短路
添加边(xi, xj, ak)
1、如果存在xi-xj < ak, 转换成xi-xj <= ak+1, 如果存在xi-xj = ak; 转换成xi-xj <= ak 和 xj-xi <= ak, 添加两条边;
2、判断解是否存在，就是判断是否存在环
3、由于图不一定连通，添加一个超级源点，到所有点连接一条边，权值为0
4、可能存在负边，所以不能用dijstra算法，可用spfa或者bellman-ford求解
