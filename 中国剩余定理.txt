中国剩余定理：
x%x1 == a && x%x2 == b && x%x3 == c;
x = n1+n2+n3
n1%x1 == a && n1%x2 == 0 && n1%x3 == 0
n2%x2 == b && n2%x1 == 0 && n2%x3 == 0
n3%x3 == b && n3%x1 == 0 && n3%x2 == 0
就求出了x
方法：
首先求出x1*x2在mod3下的逆元
然后再将其乘以a就是n1
正整数m1,m2,m3,m4,m5...mn两两互素
x = a1modm1
x = a2modm2
...
就可以按上面方法求解

扩展：
中国剩余定理要求m1,m2,m3...mn两两互素，不互素时：
x = a1+m1*x1
x = a2+m2*x2
那么a1+m1*x1 = a2+m2*x2
	m1*x1-m2*x2 = a2-a1
这样就可以利用扩展欧几里得算法extgcd求出其最小正整数解
将m1,m2,m3...mn两两合并就可以求出最小解
